5
Intent Signals & Real-Time Actions — GPT embeds these codes in its responses
🚨
Emergency Transfer
EMERGENCY_TRANSFER
AI detects imminent danger (detained family, deportation order, DV). Instantly warm-transfers to on-call attorney phone number. Urgency alert written to DB.
📅
Book Consultation
SCHEDULE_NOW
Caller is ready to book. Pulls available slots from GHL Calendar (Redis cached 1h). Books in GHL as authoritative + Google Calendar for attorney visibility. Sends SMS confirmation.
🌐
Language Switch
LANGUAGE_SWITCH_ES / EN
Caller switches language mid-call. Instantly swaps voice (Sarah↔Matilda) and system prompt (EN↔ES). Deepgram continues to auto-detect. Seamless bilingual experience.
📵
End Call
END_CALL
Graceful close triggered by AI when conversation is complete. Plays farewell message, drains TTS buffer, closes Deepgram connection, triggers post-call pipeline.
6
Post-Call Processing — Runs immediately after the call ends
🧮
Lead Scorer
GPT-4o scores lead 0–100 across 4 dimensions:
Case Value /25 Urgency /25 Readiness /25 Completeness /25
Routing: senior / associate / paralegal attorney
📊
Sentiment Scorer
GPT-4o scores call sentiment −1.0 → +1.0. Detects caller frustration level and confidence flags. Triggers escalation review if score < −0.5.
🗃️
Structured Intake Extraction
GPT-4o function-calling pass over the full transcript. Normalises free-speech answers into typed fields (dates, enums, booleans). Writes to immigration_intake table.
🔗
CRM Sync & Follow-Up
Pushes all intake fields to GHL custom fields. Tags contact with case type + urgency level. Enqueues SMS follow-up. Cost tracked (Deepgram + OpenAI + ElevenLabs).

Built-in Resilience & Fallbacks
Circuit Breaker — 3 failures in 60s trips the circuit (30s timeout). Recovers automatically. Pre-recorded filler audio ("one moment please") plays instantly during outages — no dead air.  ·  Retry with backoff — transient HTTP errors retried up to 2× with exponential backoff.  ·  Silence detection — 15s warning "Are you still there?", 30s hard timeout closes call gracefully.  ·  SIGTERM grace period — 30s drain of active WebSocket calls before shutdown.
